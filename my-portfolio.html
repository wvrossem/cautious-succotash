🚀 Sadaritagi 사다리타기

Date: June 2025

Coding Language: Scratch

🎮 About My Project

Work in progress version of the Korean Sadaritagi (사다리타기 "ladder climbing") game. It's kind of game to create random pairings. In this version a cat randomly starts at one of the four legs and will randomly end up at one of the four end points due to the randomly placed ladder legs.

It is also known in Japanese as Amidakuji (阿弥陀籤; "Amida lottery") and in Chinese as Guijiaotu (Chinese: 鬼腳圖,  "ghost leg diagram"). 

🧩 How It Works (The Logic)

The code draws the ladders by drawing the vertical lines and randomly placed horizontal lines to connect the different "legs".

It cat moves downwards and uses the green and purple dots on the connecting legs to move between the legs.

🐞 What I Want to Improve

The cat should make a combination for each starting and ending position.
