🎮 About My Project

I wanted to make a game inspired by an "Infinite Scroller" game I found on Kongregate https://www.kongregate.com/games/mjkgames/infinite-scroller

🧩 How It Works (The Logic)

I'm using a simple bar with a gap which appears at a random position on the X axis so that the gap moves around.

The player's ball is always falling and can be moved through the gaps.

🐞 What I Still Want To Improve

  • I want to add a score system and a way for the player to win or lose.
  • The ball should fall more naturally.
